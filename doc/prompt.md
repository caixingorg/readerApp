# 本地电子书阅读器应用 - 完整UI设计提示词(全功能版)

## 项目概述
创建一个功能完备的本地电子书阅读器Web应用,采用Google Material Design 3设计语言,包含完整的书籍管理、阅读、笔记、统计分析等功能模块。应用需具备专业级的交互体验和精致的视觉呈现。

## 设计理念

### 核心设计原则
- **Material Design 3风格**:动态色彩系统、圆润表面、柔和阴影、流畅动效
- **模块化架构**:每个功能模块独立且相互关联
- **渐进式体验**:从简单到复杂,引导用户探索功能
- **数据可视化**:统计数据以图表形式直观呈现
- **跨设备同步**:支持数据导入导出和云端备份

### 视觉系统
- **色彩**:主色#1A73E8(靛蓝)、辅色#34A853(绿色)、强调色#FF6F00(橙色)
- **圆角**:统一使用12px圆角,按钮16px,卡片8px
- **阴影**:4级elevation系统(2dp/4dp/8dp/16dp)
- **间距**:8px基础网格系统
- **字体**:Google Sans(标题)+ Roboto(正文)+ 可切换中文字体

### 2. 主页/书库页面
**功能**:展示所有书籍,提供快速访问和管理

**顶部区域**:
- 玻璃态导航栏:应用Logo、全局搜索、通知中心、用户头像
- 快速统计卡片:今日阅读时长、本月完成书籍、总藏书量(带动画数字滚动)
- 视图切换:网格视图/列表视图/封面墙视图

**内容区域**:
- **继续阅读**横幅:最近3本书的大卡片,显示阅读进度
- **智能分类**:
  - 未读书籍(带数量徽章)
  - 正在阅读(进度条显示)
  - 已完成(绿色对勾标记)
  - 收藏夹(星标图标)
- **书籍网格**:
  - 封面缩略图(3:4比例)
  - 书名(最多显示2行,超出省略)
  - 作者名(灰色小字)
  - 文件格式标签(PDF/EPUB/TXT)
  - 阅读进度环形图(右上角)
  - 更多操作按钮(三点图标)

**侧边栏**(桌面端):
- 分类树状菜单:所有书籍、按作者、按类别、按标签
- 自定义书单(可创建、重命名、删除)
- 存储空间使用情况图表

**底部导航**(移动端):
- 书库、导入、阅读器、统计、设置
- 选中状态带有波纹效果和颜色变化

**交互设计**:
- 书籍卡片悬停:上浮8px、阴影加深、显示快速操作菜单(编辑信息/删除/加入书单/导出)
- 长按多选:进入批量管理模式,顶部显示操作栏(全选/移动到/删除/导出)
- 拖拽排序:支持拖拽调整书籍顺序
- 右键菜单:完整操作列表
- 下拉刷新:更新书籍信息
- 空状态:显示引导插画和"导入第一本书"按钮

### 3. 导入/添加书籍页面
**功能**:多种方式导入电子书

**布局**:
- 大标题:"导入您的图书馆"
- 四种导入方式卡片(2x2网格):
  1. **本地文件**:文件夹图标,支持拖拽上传
  2. **扫描目录**:自动扫描指定文件夹
  3. **网络导入**:输入URL下载
  4. **批量导入**:ZIP压缩包解压

**本地文件导入**:
- 点击或拖拽上传区域
- 支持格式:EPUB、PDF、TXT、MOBI、AZW3
- 多文件同时上传
- 上传进度列表:
  - 每个文件显示进度条
  - 解析书籍元数据(封面、书名、作者)
  - 成功/失败状态图标
  - 失败原因提示

**扫描目录功能**:
- 选择本地文件夹
- 递归扫描所有支持格式
- 显示扫描进度(已扫描文件数/总数)
- 预览扫描结果列表
- 批量确认导入

**元数据编辑**:
- 导入后自动跳转到编辑页面
- 封面图片上传/替换
- 书名、作者、出版社、ISBN输入框
- 分类标签选择器(多选下拉)
- 自定义标签创建
- 简介文本框(支持Markdown)
- 评分(5星评分系统)

**交互设计**:
- 拖拽文件到页面任意位置触发上传
- 上传中显示全局进度提示(Snackbar)
- 自动提取元数据(从文件内部或文件名)
- 重复文件检测和提示
- 批量编辑元数据功能

4. **统计标签**:

**底部推荐区**:

**交互设计**:
- 封面点击:全屏画廊模式,支持双指缩放

### 5. 阅读器页面(核心功能)
**功能**:沉浸式阅读体验

**全屏模式**:
- 默认隐藏所有UI,纯内容显示
- 点击屏幕中央:呼出顶部和底部工具栏
- 点击屏幕左侧:上一页
- 点击屏幕右侧:下一页

**顶部工具栏**(半透明背景):
- 返回按钮(带确认,防止误触)
- 当前章节标题(滚动显示)
- 书签按钮(已添加显示实心图标)
- 更多菜单:分享、复制链接、报告问题

**底部工具栏**:
- 进度滑块:
  - 拖动时实时显示页码和百分比气泡
  - 显示章节分隔点(小圆点标记)
  - 左右箭头快速翻页
- 快速操作按钮:
  - 目录(侧边抽屉)
  - 笔记(侧边抽屉)
  - 搜索(全文搜索)
  - 设置(底部弹窗)

**左侧抽屉-目录**:
- 层级目录树
- 当前位置高亮
- 点击章节跳转
- 长按章节添加书签
- 折叠/展开子章节

**右侧抽屉-笔记与书签**:
- 双标签切换(笔记/书签)
- 笔记列表:
  - 高亮颜色分类
  - 笔记内容预览
  - 编辑/删除操作
  - 点击定位到原文
- 书签列表:
  - 页码和预览文本
  - 重命名功能
  - 删除功能

**阅读设置面板**(底部弹出):
- **字体设置**:
  - 字体选择:下拉列表+实时预览
  - 字号:滑块调节(12-32px)
  - 字重:常规/粗体切换
  - 行距:滑块调节(1.2-2.5)
  - 段距:滑块调节
  - 页边距:滑块调节
  
- **主题设置**:
  - 预设主题卡片(可左右滚动):
    - 日间(白底黑字)
    - 夜间(深灰底浅灰字)
    - 护眼(米黄底深褐字)
    - 羊皮纸(仿古风格)
    - 自定义(颜色选择器)
  - 自动切换:根据系统时间
  
- **翻页设置**:
  - 模式选择:平滑滚动/仿真翻页/淡入淡出/无动画
  - 翻页动画速度
  - 屏幕常亮开关
  
- **高级设置**:
  - 章节预加载
  - 图片显示模式
  - 禁用选择(防止误操作)
  - 音量键翻页

**文本交互**:
- **选择文本**:
  - 长按触发选择
  - 调整选择范围(拖动手柄)
  - 弹出气泡工具栏:

    - 添加笔记(弹出输入框)
    - 复制
    - 搜索(百度/Google)
    - 翻译(英汉互译)
    - 朗读(TTS)
    - 分享

**手势操作**:
- 双指捏合:调整字体大小
- 三指左滑:下一章
- 三指右滑:上一章
- 双击屏幕:呼出/隐藏工具栏
- 屏幕左侧边缘滑动:调节亮度
- 屏幕右侧边缘滑动:调节进度

**全文搜索功能**:
- 搜索框(顶部下拉)
- 实时搜索结果列表
- 高亮所有匹配项
- 上一个/下一个导航
- 搜索历史记录

**朗读功能**(TTS):
- 播放/暂停按钮
- 语速调节
- 音色选择
- 自动翻页
- 定时停止

**交互设计**:
- 所有UI元素淡入淡出动画
- 翻页动画流畅(60fps)
- 触摸反馈haptic震动
- 工具栏自动隐藏(3秒无操作)
- 阅读进度自动保存(每30秒)
- 断点续读(下次打开自动定位)

### 7. 统计分析页面
**功能**:可视化展示阅读数据和习惯

**概览仪表盘**:
- **顶部四大指标卡片**:
  1. 总阅读时长(小时,大数字显示)
  2. 完成书籍(本,带同比增长)
  3. 阅读天数(连续打卡天数)
  4. 平均阅读速度(字/分钟)

**详细图表区域**:

1. **阅读趋势折线图**:
   - X轴:时间(可切换:周/月/年)
   - Y轴:阅读时长(分钟)
   - 多条曲线对比不同书籍
   - 悬停显示具体数值
   - 可点击图例隐藏/显示曲线

2. **阅读时段热力图**:
   - X轴:24小时
   - Y轴:周一到周日
   - 颜色深浅表示活跃度
   - 找出最佳阅读时段

3. **书籍分类饼图**:
   - 按类别统计阅读时长占比
   - 点击扇区查看该类别书籍列表
   - 悬停显示百分比和时长

4. **阅读进度条形图**:
   - 每本书的完成进度对比
   - 按进度排序
   - 颜色区分:已完成(绿)、进行中(蓝)、未开始(灰)

5. **阅读目标进度环**:
   - 设定年度/月度阅读目标
   - 环形图显示完成度
   - 剩余天数倒计时
   - 建议每日阅读时长

**成就系统**:
- 徽章墙(网格布局):
  - 首次阅读
  - 完成10本书
  - 连续阅读7天
  - 夜猫子(凌晨阅读)
  - 早起鸟(清晨阅读)
  - 速读者(阅读速度超快)
  - 笔记达人(笔记超100条)
  - 收藏家(藏书超50本)
- 未解锁徽章灰色显示
- 点击徽章查看获取条件

**排行榜**:
- 本月最爱书籍Top10
- 最爱作者Top5
- 阅读最快书籍
- 笔记最多书籍

**时间轴视图**:
- 按时间倒序显示所有阅读活动:
  - 开始阅读某书
  - 完成某书
  - 添加笔记
  - 解锁成就
- 每个事件带有图标和时间戳

**导出报告**:
- 生成年度/月度阅读报告
- PDF格式,美观排版
- 包含所有图表和统计数据
- 分享到社交媒体

**交互设计**:
- 图表可交互(缩放、拖拽、筛选)
- 数字滚动动画
- 图表加载骨架屏
- 时间范围选择器(日历组件)
- 响应式图表(移动端优化)
